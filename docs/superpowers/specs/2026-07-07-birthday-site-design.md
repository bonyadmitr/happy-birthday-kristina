# Дизайн: сайт-поздравление для Кристины

**Дата:** 2026-07-07
**Деплой:** GitHub Pages, репозиторий `happy-birthday-kristina`

## Цель

Красивый интерактивный сайт-поздравление с днём рождения. Приоритет — безупречный вид и
поведение на iPhone (Safari). Личный контент (фото, текст, музыка) добавляется позже в
понятные места без правки логики.

## Стек и архитектура

- Чистые **HTML + CSS + JS**, без фреймворка и без билд-шага (архетип «vanilla, iPhone-first»).
- Один `index.html`, грузится браузером напрямую → тривиальный деплой на GitHub Pages.
- Файлы:
  - `index.html` — разметка секций.
  - `css/styles.css` — стили + палитры тем в CSS-переменных.
  - `js/config.js` — **вся персонализация** (имя, тексты, ссылка на музыку, список фото).
  - `js/confetti.js` — вшитый локально `canvas-confetti` (без внешнего CDN).
  - `js/main.js` — логика сцен, интерактив, конфетти-вызовы.
  - `assets/photos/`, `assets/audio/` — плейсхолдеры под фото и mp3.
  - `README.md` — инструкция по деплою и персонализации.

## Сцена (одна страница, сверху вниз)

1. **Распаковка подарка** — коробка с бантом на весь экран; тап (или тряска телефона)
   «распаковывает» → взрыв конфетти → оверлей уходит, открывается сайт. Тап-жест решает
   проблему автоплея аудио на iOS.
2. **Hero** — «С днём рождения, Кристина», тёплый градиент, плавающие шарики/сердечки на фоне.
3. **Тортик со свечами** — тап «задувает» свечи → салют конфетти → проявляется тёплая надпись.
4. **Галерея / таймлайн воспоминаний** — карточки, появляются при скролле (native
   scroll-driven анимации; fallback на IntersectionObserver). Пока стильные плейсхолдеры.
5. **Письмо** — раскрывающееся личное поздравление (тёплая заготовка под правку).
6. **Финал** — «Я тебя люблю» + мягкое конфетти, кнопка «повторить сначала».

## Темизация (ключевое требование — легко менять)

- Все цвета/шрифты — в CSS-переменных на `:root`.
- 4 палитры через атрибут `data-theme` на `<html>`: `romantic` (по умолчанию), `elegant`,
  `playful`, `dreamy`.
- Переключение: сменить `data-theme` в HTML **или** открыть `?theme=elegant` в адресе.
- Маленький скрытый переключатель тем для быстрого сравнения глазами.

## Персонализация (потом, без правки логики)

- `js/config.js` — единая точка: `name`, тексты hero/торта/письма/финала, `musicUrl`, массив `photos`.
- Фото кладутся в `assets/photos/`, mp3 — в `assets/audio/`; пути прописываются в config.
- Пустые/отсутствующие поля → показываются аккуратные плейсхолдеры, сайт не ломается.

## iPhone-first

- `env(safe-area-inset-*)` — учёт чёлки/домашней полоски.
- `dvh` вместо `vh` — не прыгает при скрытии адресной строки Safari.
- Крупные тач-зоны (≥44px), никаких hover-only взаимодействий.
- Аудио стартует только по пользовательскому тапу (iOS-политика автоплея).
- `prefers-reduced-motion` — отключение тяжёлых анимаций, конфетти дозируется.

## Интерактив и конфетти

- Конфетти: локальный `canvas-confetti` (вшит в репо).
- Триггеры: распаковка подарка, задувание свечей, финал, кнопка «повторить».
- Плавающие частицы (шарики/сердечки) — лёгкий CSS/JS-слой на фоне, реагирует на тап;
  опционально — на наклон телефона (DeviceOrientation, только по согласию, с фолбэком).

## Проверка (по докам, web-стек)

- Локальный статический сервер + прогон в браузере, эмуляция iPhone (узкий вьюпорт,
  safe-area, dark/light).
- Проверка консоли на ошибки, аудио-gate по тапу, поведение при `prefers-reduced-motion`.

## Вне scope (YAGNI)

- Бэкенд, RSVP, сбор данных гостей, аналитика.
- Сборка/бандлер, фреймворки, внешние CDN в рантайме.
